# personal-website

Static personal site for Harshal Rudra, deployed on GitHub Pages.

- `index.html` — home page
- `post-*.html` — blog posts
- `support.js` — runtime (loads React/Babel/KaTeX from CDN at runtime)
- `*.png` / `*.jpg` / `*.gif` — blog images, referenced relatively

No build step. GitHub Pages serves the repo root of `main` directly.
Live at https://theharshal30.github.io/personal-website/
